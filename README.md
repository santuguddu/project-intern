Project Setup and Execution

Follow the steps below to set up and run the project locally.

1. Install the Project

Download the project1.zip file.

Extract the contents to your desired location.

2. Setup Frontend

Open a terminal and navigate to the frontend directory:

cd frontend


Install the required dependencies:

npm install


Start the frontend development server:

npm run dev

3. Setup Backend

Open a terminal and navigate to the backend directory:

cd backend


Initialize the project (if not already initialized):

npm init -y


Install all necessary dependencies for the backend:

npm install <package-names>


Start the backend server:

npm start

4. Database

This project uses MongoDB Atlas.

Ensure your MongoDB Atlas URI is correctly configured in the backend environment variables before running the server.
